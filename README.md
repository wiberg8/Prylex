# Prylex

**Prylex** is a searchable inventory system focused on digital devices like computers, phones, tablets, and more. The application is primarily developed for schools and organizations that want to track who has which device ‚Äì whether it's a PC, iPad, or mobile phone.

- Keeps track of which person has which digital device ‚Äì e.g., computer, tablet, phone.
- Facilitates the management of large amounts of equipment.
- Eliminates the need for messy Excel spreadsheets.
- Makes inventory and troubleshooting faster and easier using barcodes.
